# Electron Calculator 

Um simples projeto de calculadora utilizando eletron.js personalizada com CSS;

# Modo de uso

Após clonar o repositório, utilize **npm i** para instalar todas as dependências, após a instalação, utilize: **npm start**;

# Stacks utilizadas:

<p> HTML5 </p>
<p> CSS3 </p>
<p> JavaScript </p>
<p> Electron </p>
